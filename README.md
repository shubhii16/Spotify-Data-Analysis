🎧 Spotify Data Analysis & Interactive Dashboard
📌 Overview

This project presents an end-to-end data analysis of Spotify streaming data, aimed at extracting meaningful insights into music trends, artist performance, and audio feature patterns. The analysis is performed using Python, followed by the development of an interactive Power BI dashboard for visual storytelling.

🎯 Problem Statement

With the rapid growth of music streaming platforms, understanding listener preferences and track performance has become crucial. This project aims to analyze Spotify data to identify key factors influencing song popularity and uncover patterns across various audio features.

📊 Objectives
Identify top-performing tracks and artists
Analyze the distribution of audio features (danceability, energy, tempo, etc.)
Discover relationships between musical attributes and popularity
Build an interactive dashboard for intuitive data exploration

📂 Dataset
Source: Spotify dataset
Features include:
Track Name
Artist
Popularity Score
Danceability
Energy
Tempo
Duration

🛠️ Tech Stack
Category	Tools Used
Programming	Python
Data Analysis	Pandas, NumPy
Visualization	Matplotlib, Seaborn
Dashboard	Power BI
Environment	Jupyter Notebook

🔍 Methodology
1. Data Preprocessing
Handled missing values using appropriate techniques
Removed duplicate records
Converted data types for accurate analysis
2. Exploratory Data Analysis (EDA)
Distribution analysis of popularity scores
Identification of top artists and tracks
Feature-wise statistical summaries
3. Feature Analysis
Correlation analysis between audio features
Impact of energy, tempo, and danceability on popularity
4. Data Visualization
Bar charts for top artists and tracks
Scatter plots for feature relationships
Heatmap for correlation matrix

📊 Dashboard

The Power BI dashboard provides an interactive interface to explore insights:

🎵 Top Tracks & Artists
📈 Popularity Trends
🎚️ Audio Feature Distribution
🔍 Filters for dynamic exploration

📈 Key Insights
Tracks with higher energy and danceability tend to have better popularity scores
A strong positive correlation exists between energy and loudness
Certain artists consistently appear among top-performing tracks
Moderate tempo tracks show higher listener engagement

✅ Conclusion

The analysis highlights how audio features influence song performance and listener preferences. By combining Python-based analysis with an interactive dashboard, the project demonstrates a practical approach to data-driven decision-making in the music industry.
